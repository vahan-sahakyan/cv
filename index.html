PROJECTS

MAPTY

Workout Application

Mapty is a Workout Mapping Web-App. It saves your workouts and then logs them on the map in ONLY 3 STEPS
1. Click on the map to register the workout location, which will trigger a workout log to appear on the left.
2. Select the type of your activity (cycling or running).
3. Fill in the required details about your workout such as distance, time, cadence (steps/min) if running or elevation gain if cycling. The pace/speed will be calculated automatically.

  • Mapty is built using the architecture of "Object Oriented Programming" [OOP] Paradigm.
  • Its logic is implemented using 2 classes:
    "Running" and "Cycling"
  • Both inherit from a base class named "Workout".

E-PIANO

Musical Intrument Application

E-Piano is a Music-Keyboard Emulator inside your browser. It allows you to play beautiful chords and melodies, without even needing an additional Midi-Controller keyboard attached. All you'll really need is a regular typing keyboard in front of your PC.

  • E-Piano is created using only the built-in Web Audio API of JavaScript.
  • The code is conducted in a modular way, and the architecture is OOP.
  • The most comfortable/playable environment will be the Desktop Browser, although it's also possible to run inside a Mobile Browser.
